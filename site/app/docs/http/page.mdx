# HTTP server

The `runtime:http` module provides a fast, built-in HTTP server using standard
web `Request` and `Response` objects.

## Usage

```js
import { serve } from "runtime:http";

// serve(options, handler) — the handler takes a Request, returns a Response.
serve({ port: 8080 }, (req) => {
  return new Response("Hello from esrun!");
});

console.log("Server listening on port 8080");
```

## Usage with Hono

Because esrun uses standard Web APIs, popular frameworks like Hono work out of
the box.

```js
import { serve } from "runtime:http";
import { Hono } from "hono";

const app = new Hono();
app.get("/", (c) => c.text("Hello from Hono on ESRun!"));

serve({ port: 8080 }, app.fetch);

console.log("Hono listening on port 8080");
```
